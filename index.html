Brendan Eich - Inventor of JavaScript

Creator of JavaScript & CEO @ Brave Software

Brendan Eich is an american technologist. He created the JavaScript Programming language in only TEN DAYS! The intention of JavaScript was to create a fully functional programming language that would run in the browser. This was the first of it's kind, and opened the door to significant development of web technologies that we know today. The original JavaScript engine was code named 'SpiderMonkey' and developed for the Netscape Navigator browser in 1995. Brendan co-founded Mozilla and eventually moved on to become CEO of Brave Software.

More About Eich

Brendan received a bachelor's degree in mathematics and computer science at Santa Clara University. He received a Master's Degree in 1985 from the Universiy of Illinois at Urbana-Champaign. Brendan Eich co-founded Mozilla in 1998, creating a site to manage open-source contributions to the Netscape source code. He served as Mozilla's chief Architect.

JavaScript / ECMAScript

The original name for JavaScript was 'Mocha' chosen by Marc Andreessen founder of Netscape. It was quickly changed to LiveScript and was changed again later when receiving a trademark from Sun - receiving it's most common name: JavaScript. In 1996-1997 Javascript was taken to ECMA (European Computer Manufacturer's Associan) to carve out standard specification to implement it's use. This work done at ECMA eventually led to the official release of ECMA-262. ECMAScript is the name of the official standard, with JavaScript being the most well known of the implementations. ActionScript 3 is another well-known implementation of ECMAScript.

SpiderMonkey

  The first JavaScript engine was code-named SpiderMonkey and developed over two weeks by Eich as he rewrote the original Mocha codebase. Shortly afterwards the first JIT or just-in-time compiler was writter for the JavaScript language. The compiler was released as part of SpiderMonkey in Firefox version 3.5. Instead of compiling whole functions, TraceMonkey was a tracing JIT which operates by recording control flow and data types during interpreter execution. This data then informed the construction of trace trees -highly specialized paths of native code. SpiderMonkey was present in Firefox until version 11, which it was replaced by JägerMonkey.

Career

After making major contributions to Netscape, Firefox, and JavaScript- Eich was promoted to CEO of Mozilla in March 2014. Gary Kovacs, John Lily, and Ellen Siminoff resigned from the Mozilla board after the appointment of Eich as CEO. Shortly after being appointed as CEO Eich found himself amidst a firestorm. Eich had donated to Proposition 8, a failed California ballot initiative to maintain that only man/woman marriages be legal. When the news of this donation became public he was highly criticised, and ultimately resigned from the role as CEO.

What's he doing now?

Eich is now launching a new startup, called Brave, that aims to "fix" a problem with modern internet - Advertisements. Eich is developing a new web browser that blocks ALL ads by default ensuring speed and privacy. By stripping out all of the extra junk that loads every time you click speed will drastically improve.

Web Assembly

Brendan and Brave has also become involved in the Web Assembly project which boosts web performance by providing a portable code format to run in browsers at native speeds. Major browser vendors Apple, Google, and Mozilla are all on-board with the effort. The technology serving as a binary syntax, started with the asm.js subset of JavaScript. At some point the Web Assembly becomes the safe native format for the web. It will help bring the web equal to native applications in terms of performance.
